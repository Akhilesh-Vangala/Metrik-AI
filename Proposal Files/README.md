# Metrik AI

**Building Energy Prediction and Anomaly Detection**

DS-GA 1019 Advanced Python for Data Science · Spring 2026

---

## Project

Metrik AI is a scalable ML-powered pipeline for building energy: **forecasting** (next-hour consumption), **anomaly detection** (residual-based), and **decision support** (ranked audit list). Dataset: ASHRAE Great Energy Predictor III (53.6M hourly readings, 1,636 buildings).

## Folder structure

- **Proposal Files/** — All proposal and overview documents (LaTeX, HTML, markdown), flowchart, dataset notes, architect spec, checklist, and data download script.

Rename this folder to **Metrik AI** on your Desktop if you want the project to appear as "Metrik AI".
